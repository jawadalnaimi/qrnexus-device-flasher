QR Nexus
Alarm Device — 1‑click Installer
⚠️ Use Google Chrome or Microsoft Edge on a computer (laptop/desktop). It does not work in Safari, Firefox, or on a phone.

This installs the QR Nexus alarm firmware onto an ESP32 board, straight from your browser — nothing to download. Afterwards the board creates its own WiFi where you type in your network and device code. No technical knowledge needed.

This browser can't do it — open this page in Chrome or Edge. Please allow the USB/serial port when asked.
  1. Plug the ESP32 board into your computer with a USB cable (a real data cable, not a charge‑only one).
  2. Click ⚡ Install firmware. A small window lists ports — pick the one that appears (often “USB Serial” or “CP210x / CH340”). Then choose Install.
  3. Wait about a minute for it to finish, then unplug/replug or press the board's reset button.
  4. On your phone or laptop WiFi list, join the network QRNexus‑Setup‑XXXX and open http://192.168.4.1.
  5. Type your WiFi name + password and your device code (qrn_dev_…, from app.qrnexus.io → Alarm Panels → key icon), then Save. Done — the board connects and starts reporting.
The board doesn't show up in the port list?

Your computer may need a tiny USB driver for the board's chip:
• CP210x driver: silabs.com
• CH340/CH341 driver: wch-ic.com
Then try a different USB cable or port, and reload this page. (macOS/Windows usually already have these.)

Installer says "Failed to initialize", or the browser crashes?
Some ESP32 boards can't be reset over the browser. Use the downloadable flasher instead — same firmware, more reliable (no browser, no BOOT button):

⬇︎ Download for macOS ⬇︎ Download for Windows

Unzip, then double-click Flash-macOS.command / Flash-Windows.bat. First run (it's unsigned): macOS → System Settings → Privacy & Security → "Open Anyway"; Windows → "More info" → "Run anyway". (Mac shortcut: in Terminal, xattr -cr the unzipped folder.)

Need it even simpler? Ask whoever set this up to flash & configure one board for you, then just plug it in.
